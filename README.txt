/Como compilar e rodar o programa:                                                                                 /
/                                                                                                                  /
/   Sistema operacional recomendado: Windows 10                                                                    /
/   -Abra o arquivo “ICC.c” com o CodeBlocks e com o respectivo compilador;                                        /
/   -Clique em “Build and run” (atalho: botão F9);                                                                 /
/   -O arquivo .exe será compilado e logo executado.                                                               /
/                                                                                                                  /
/Vale ressaltar que as pastas “Alunos” e "Artes", com seus respectivos arquivos de texto, devem estar, junto com o /
/executável, em um mesmo diretório do computador. Isso garante o correto funcionamento do projeto.                 /
